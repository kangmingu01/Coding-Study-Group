## 백준 단계별문제풀이(C++)

-----
- 문제 : https://www.acmicpc.net/problem/2480
- 코드
```C++
#include <iostream>
using namespace std;

int main()
{
	int a, b, c;
	cin >> a >> b >> c;
	if (a == b && a == c)
	{
		cout << 10000 + a * 1000 << endl;
	}
	else if (a == b || a == c)
	{
		cout << 1000 + a * 100 << endl;
	}
	else if (b == c)
	{
		cout << 1000 + b * 100 << endl;
	}
	else
	{
		cout << max(a, max(b, c)) * 100 << endl;
	}
}
```
